Etape 5 : Décrypter le mot de passe

Encore une fois le mot de passe semble crypté

Cette fois-ci je ne sais pas comment le decrypter, cependant il s'agit probablement d'un cryptage connu, peut être pouvons nous essayer plusieurs solutions pour le décrypter en espérant que le mot de passe de ce méchant ne soit pas trop sécurisé...

Rendez-vous sur https://md5hashing.net/hash, dans la partie verte , copier-coller le mot de passe dans Hash String. Ensuite essayer les différents algorithme de décryptage présent dans la liste Hash Type et appuyer sur Decode jusqu'a trouver le mot de passe.

Ensuite vous pouvez saisir l'identifiant et le mot de passe pour désamorcer la bombe !
