Etape 2 : Déchiffrer l'identifiant

Aie, il semblerait que le mot de passe trouvé soit crypté !

Heureusement, je sais comment le déchiffrer, il faut utiliser le code de César, c'est super simple !

Pour chaque lettre de l'identifiant :

On prend la lettre

On transforme la lettre en chiffre en fonction de sa position dans l'alphabet (A=1, B=2...)

On enlève 3 à ce chiffre

On retransforme le chiffre en lettre, tadaaaa c'est tout de suite plus lisible !

Mais ça semble un peu long à faire à la main..., trouve un moyen d'afficher le code source de la page. Dans le code source, tu devrais trouver un moyen de déchiffrer cet identifiant plus rapidement
