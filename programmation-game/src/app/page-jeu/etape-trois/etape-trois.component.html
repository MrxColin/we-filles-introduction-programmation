Etape 3 : Trouvez le mot de passe

Malheureusement pour moi, vous progressez vite... Vous êtes maintenant en possession de mon identifiant, mais il vous manque encore le mot de passe !

Vous aurez peut-être remarqué un bouton très frustrant sur lequel vous ne pouvez cliquer, mais qui semble pourtant si prometteur...

Comme pour l'étape 1, inspectez la page. Cette fois-ci, intéressez-vous aux Elements de la page. Vous y trouverez le code HTML de la page, qui décrit tous les éléments présents sur la page.

Dépilez les éléments à l'aide des flèches sur la gauche et parcourez-les jusqu'à trouver le bouton "Afficher le mot de passe".

Une fois le bouton trouvé, vous devriez voir le mot-clé "disabled" au niveau de ce bouton, effacez-le et le bouton devrait fonctionner !
